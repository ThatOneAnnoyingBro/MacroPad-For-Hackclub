firmware based, they are all made via https://qmk.fm/ here. And i also used VIA so i can easily change the settings of the keys. Right now its set as:

SW1 → Play/Pause, SW2 → Mute, SW3 → Vol Down, SW4 → Vol Up, SW5 → Stop
Encoder 1 → Volume up/down when rotating
Encoder 2 → Scroll up/down when rotating

But it shall be changed later...

# BOM: 
* Seeed XIAO RP2040 (U1)
* Rotary Encoders x2 (SW9, SW10) — Alps EC11E footprint
* Tactile switches x5 (SW1-SW5)
* 0.91" OLED display (I2C, SSD1306, 128x32)
* 3D printed case x1
